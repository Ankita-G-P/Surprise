### ✨ Adding Photos & Videos
You can now add both photos and videos!
- For **Timeline**: Update `CONFIG.timelineData` in `script.js`. Set `mediaType: "video"` for videos.
- For **Gifts**: Update `CONFIG.giftData` in `script.js`. Set `type: "video"` for videos in the `media` array.
- Supported extensions: `.jpg`, `.png`, `.mp4`, `.webm`.
